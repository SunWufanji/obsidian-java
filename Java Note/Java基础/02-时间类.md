# Java 日期时间类核心要点
## Date 类

- java.util.Date：日期 + 时间，常用` getTime ()`、`before/after ()`
- java.sql.Date：数据库专用，只有日期，继承自 `util.Date`

## Calendar

- 抽象类，通过 `getInstance ()` 获取对象
- 用于获取 / 设置年月日、时间计算
## Java 8 新日期类（推荐）

- LocalDate：只日期
- LocalTime：只时间
- LocalDateTime：日期 + 时间
- 特点：不可变、线程安全
- 常用：plus/minus/with/isBefore/isAfter

## 日期格式化

- SimpleDateFormat：配合 Date，非线程安全
- DateTimeFormatter：配合 Java 8 日期类，线程安全

## 时间戳

- 1970-01-01 至今的毫秒数
- 获取：System.currentTimeMillis ()
- 可与 Date 互相转换

## 使用选择

- 旧项目：Date + SimpleDateFormat
- 日期计算：Calendar
- 生产 / 多线程：Java 8 新日期类
- 数据库：java.sql.Date
