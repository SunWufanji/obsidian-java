# 54. 螺旋矩阵

## 题目

给你一个 `m x n` 的矩阵 `matrix` ，请按照顺时针螺旋顺序，返回矩阵中的所有元素。

例如：

```text
1 2 3
4 5 6
7 8 9
```

输出：

```text
[1, 2, 3, 6, 9, 8, 7, 4, 5]
```

---

## 题意理解

这题不是修改矩阵，而是按顺时针方向一圈一圈遍历矩阵，把经过的元素按顺序放入结果列表。

遍历顺序固定为：

1. 从左到右
2. 从上到下
3. 从右到左
4. 从下到上

每走完一圈，就把边界向内收缩，继续处理下一圈。

---

## 核心思路

用四个边界表示当前这一圈还能走的范围：

- `top`：上边界
- `bottom`：下边界
- `left`：左边界
- `right`：右边界

每一轮依次遍历：

1. 上边：`left -> right`
2. 右边：`top -> bottom`
3. 下边：`right -> left`
4. 左边：`bottom -> top`

每走完一条边，就把对应边界向内收缩一格。

---

## 标准解法

### Java 代码

```java
import java.util.ArrayList;
import java.util.List;

class Solution {
    public List<Integer> spiralOrder(int[][] matrix) {
        List<Integer> ans = new ArrayList<>();
        if (matrix == null || matrix.length == 0 || matrix[0].length == 0) {
            return ans;
        }

        int top = 0;
        int bottom = matrix.length - 1;
        int left = 0;
        int right = matrix[0].length - 1;

        while (top <= bottom && left <= right) {
            for (int j = left; j <= right; j++) {
                ans.add(matrix[top][j]);
            }
            top++;

            for (int i = top; i <= bottom; i++) {
                ans.add(matrix[i][right]);
            }
            right--;

            if (top <= bottom) {
                for (int j = right; j >= left; j--) {
                    ans.add(matrix[bottom][j]);
                }
                bottom--;
            }

            if (left <= right) {
                for (int i = bottom; i >= top; i--) {
                    ans.add(matrix[i][left]);
                }
                left++;
            }
        }

        return ans;
    }
}
```

## 执行过程演示

以矩阵：

```text
1 2 3
4 5 6
7 8 9
```

初始边界为：

- `top = 0`
- `bottom = 2`
- `left = 0`
- `right = 2`

第一圈：

- 上边：`1 2 3`
- 右边：`6 9`
- 下边：`8 7`
- 左边：`4`

结果为：

```text
[1, 2, 3, 6, 9, 8, 7, 4]
```

然后边界缩小为：

- `top = 1`
- `bottom = 1`
- `left = 1`
- `right = 1`

第二圈只剩中间元素 `5`，加入后最终结果为：

```text
[1, 2, 3, 6, 9, 8, 7, 4, 5]
```

## 为什么要加判断

在遍历下边之前，要先判断：

```java
if (top <= bottom)
```

在遍历左边之前，要先判断：

```java
if (left <= right)
```

原因是有些情况下矩阵只剩一行或一列，如果继续走下边或左边，就会重复加入元素。

---

## 时间复杂度

矩阵一共有 `m * n` 个元素。

整个过程中：

- 每个元素只会被访问一次
- 不会重复遍历同一个元素

所以时间复杂度是：

```text
O(m * n)
```

---

## 空间复杂度

如果不算返回结果列表，只使用了四个边界变量，因此额外空间复杂度是：

```text
O(1)
```

如果把结果列表也算进去，因为结果中要存 `m * n` 个元素，所以是：

```text
O(m * n)
```

刷题时通常说额外空间复杂度为 `O(1)`。

---

## 易错点

### 1. 忘记边界判断

如果少了：

```java
if (top <= bottom)
if (left <= right)
```

就可能在单行或单列时重复访问元素。

### 2. 边界更新顺序写乱

要记住：

- 走完上边，`top++`
- 走完右边，`right--`
- 走完下边，`bottom--`
- 走完左边，`left++`

### 3. 行列不要混淆

- `matrix[i][j]`
- `i` 表示行
- `j` 表示列

---

## 暴力 / 其他解法

### 解法二：方向数组 + `visited`

也可以准备四个方向：

- 右
- 下
- 左
- 上

每次沿当前方向前进，如果下一步越界或者已经访问过，就切换方向。

这种方法也能做出来，但需要额外的 `visited[m][n]` 数组，因此空间复杂度是：

```text
O(m * n)
```

所以这题更推荐掌握边界收缩法。

## 面试表述

可以这样说：

> 我用四个边界 `top、bottom、left、right` 来模拟一圈一圈遍历矩阵。  
> 每轮按顺时针依次遍历上边、右边、下边、左边。  
> 每走完一条边，就把对应边界向内收缩一格。  
> 为了避免单行或单列时重复访问，我在遍历下边和左边前加了边界判断。  
> 每个元素只访问一次，所以时间复杂度是 `O(mn)`，额外空间复杂度是 `O(1)`。

## 本题总结

这题最核心的是：

- 用四个边界控制当前还能走的范围
- 按上、右、下、左的顺序一圈一圈遍历
- 遍历完成后不断收缩边界

重点记住：

1. `while (top <= bottom && left <= right)`
2. 遍历下边前判断 `top <= bottom`
3. 遍历左边前判断 `left <= right`
4. 每个元素只访问一次，所以时间复杂度为 `O(m * n)`

```java
import java.util.ArrayList;
import java.util.List;

class Solution {
    public List<Integer> spiralOrder(int[][] matrix) {
        List<Integer> ans = new ArrayList<>();
        int m = matrix.length;
        int n = matrix[0].length;

        int top = 0;
        int bottom = m - 1;
        int left = 0;
        int right = n - 1;

        while (top <= bottom && left <= right) {
            List<Integer> ring = new ArrayList<>();

            // 当前这一圈的四个角坐标
            int leftTopRow = top, leftTopCol = left;
            int rightTopRow = top, rightTopCol = right;
            int leftBottomRow = bottom, leftBottomCol = left;
            int rightBottomRow = bottom, rightBottomCol = right;

            // 只剩一行
            if (top == bottom) {
                for (int j = left; j <= right; j++) {
                    ring.add(matrix[top][j]);
                }
            }
            // 只剩一列
            else if (left == right) {
                for (int i = top; i <= bottom; i++) {
                    ring.add(matrix[i][left]);
                }
            }
            // 正常走一圈
            else {
                // 左上 -> 右上
                for (int j = leftTopCol; j <= rightTopCol; j++) {
                    ring.add(matrix[leftTopRow][j]);
                }

                // 右上下面一个 -> 右下
                for (int i = rightTopRow + 1; i <= rightBottomRow; i++) {
                    ring.add(matrix[i][rightTopCol]);
                }

                // 右下左边一个 -> 左下
                for (int j = rightBottomCol - 1; j >= leftBottomCol; j--) {
                    ring.add(matrix[rightBottomRow][j]);
                }

                // 左下上面一个 -> 左上下面一个
                for (int i = leftBottomRow - 1; i > leftTopRow; i--) {
                    ring.add(matrix[i][leftBottomCol]);
                }
            }

            ans.addAll(ring);

            // 整体缩小一圈
            top++;
            bottom--;
            left++;
            right--;
        }

        return ans;
    }
}

```
