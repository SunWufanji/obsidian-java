# 助手工作规则（算法题）

> 算法题整理助手的工作规则，忘了来看这里。

---

## 一、文件命名

`XX. 题目名.md`

示例：`24. 两两交换链表中的节点.md`

## 二、仓库路径

`D:\Javanote\Java Note\算法题\[分类]\`

当前分类：`链表\`、`数组\`、`哈希表\`

## 三、笔记结构

```
# XX. 题目名（一级标题）

## 题目
原题描述 + 示例

## 题意理解
用自己的话翻译一遍

## 方法一：XXX
- 核心思路
- 完整 Java 代码
- 图解 / 执行过程（每一步的链表/数组变化）

## 方法二：XXX
...
（列出所有解法）

## 三种方法对比
对比表

## 做题建议
先掌握哪个、面试推荐哪个

## 易错点

## 本题总结
```

## 四、核心要求

1. **每个解法都要写出来**
2. **最优解**做最详细解释（时间复杂度/空间复杂度推导 + 每一步执行过程图解）
3. 代码要有，图解要有，对比要有

## 五、文件已存在时

如果文件已创建好，直接填充内容，不要重新创建。
